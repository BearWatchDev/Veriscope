◆

Advanced Multi-Layer Deobfuscation

27 specialized decoders unwrap up to 6 layers of nested obfuscation automatically. Handles Base64 variants, GZIP/zlib/bzip2 compression, hexadecimal, string reversal, ROT13, multi-byte XOR patterns, URL encoding, HTML entities, PowerShell UTF-16LE, and JavaScript wrappers. SHA1-based cycle detection prevents infinite loops.

◆

Intelligent Quality Detection

Quality regression system tracks decoding quality at each layer with 45% word frequency, 25% letter ratio, 20% structure analysis, and 10% encoding penalty weighting. Automatically stops at quality peaks to prevent over-decoding. Detects 45+ malware-specific keywords for accurate plaintext identification.

◆

Comprehensive IOC Extraction

Extracts actionable indicators from both original and deobfuscated content with regex-based pattern matching: URLs, IPv4 addresses, domains, email addresses, Windows file paths, registry keys, mutex names, and cryptocurrency wallet addresses (BTC/ETH/XMR). High-entropy strings (40+ chars) automatically flagged for encoded payloads.

◆

MITRE ATT&CK Mapping

Automatic technique mapping with confidence scoring across 8 major tactics: persistence, privilege escalation, defense evasion, credential access, discovery, lateral movement, command & control, and exfiltration. Interactive modal interface groups findings by tactic with technique IDs and direct links to MITRE documentation.

◆

Real-Time Progress Tracking

Server-Sent Events (SSE) stream live deobfuscation progress with animated decode chains showing method sequences (e.g., "base64 → hex → rot13 → xor"). Clickable summary cards display layer-by-layer quality scores. Progress bar with granular decimal updates (e.g., layer 2.3 = 30% through layer 2 methods).

◆

YARA Rule Generation

File-based detection signatures include both obfuscated and deobfuscated strings for multi-stage coverage. High-entropy strings auto-added for encoded payload detection. Comprehensive metadata with ATT&CK technique mappings, TLP classification, sample SHA256 hashes, and severity ratings. Generates both catch-all and individual IOC-specific rules.

◆

Sigma Rule Generation

SIEM-compatible detection rules in YAML format covering network activity (URLs, IPs), process execution (commands, scripts), registry modifications, and file operations. Compatible with 50+ SIEM platforms including Splunk, Elastic, QRadar, ArcSight, and Microsoft Sentinel. Includes severity levels and operational metadata.

◆

Modular Preset System

Five configurable presets optimize for different scenarios: conservative (strict validation), balanced (default), aggressive (deeper decoding), malware_analysis (real-world samples), and research-optimized (complex multi-layer). Automatic preset rotation tries multiple strategies when default fails, selecting best result by quality score.

◆

Comprehensive Reporting

Multi-format output includes Markdown reports (human-readable analysis), JSON (machine-readable full results), YARA rules (file-based detection), and Sigma rules (SIEM integration). Detailed trace logs show method chains with success/failure indicators. Web GUI provides tabbed interface with downloadable artifacts for each output type.

Important Information

Static Analysis Only

Veriscope performs text-based analysis exclusively. No code execution, sandboxing, or dynamic analysis. Safe for production workstations. All processing happens through pattern matching and decoding algorithms—malware samples are never executed.

Detection Rules are Templates

Generated YARA/Sigma rules require human review and validation before production deployment. Always test in controlled environments first. Rules provide a strong starting point but should be tuned for your specific environment and threat model.

ATT&CK Mapping Limitations

Technique mappings are heuristic-based using keyword matching against MITRE's framework. Correlate with additional threat intelligence, sandbox analysis, and behavioral telemetry for accurate attribution. Confidence scores indicate reliability of each mapping.

Responsible Use

Veriscope is built exclusively for defensive security operations. Use it to analyze threats, build detections, protect networks, and accelerate incident response. Always ensure compliance with applicable laws and regulations. This tool is designed to empower security teams—not threat actors.
